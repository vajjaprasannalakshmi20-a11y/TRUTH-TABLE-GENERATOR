Resume Builder Pro (Python GUI - Tkinter)

📌 Project Description

Resume Builder Pro is a Python GUI application built using Tkinter.
It allows users to enter personal and professional details and automatically generates a structured resume.

The app also provides an option to save the resume as a text file (.txt).


---

✨ Features

👤 User details input (Name, Email, Phone)

🏠 Address input

🎓 Education section

🧠 Skills section

💼 Experience section

📄 Automatic resume generation

💾 Save resume as .txt file

🖥️ Simple and clean GUI interface



---

🛠️ Technologies Used

Python 3 🐍

Tkinter (GUI Library)

🚀 How to Run in VS Code

1. Install Python

Download from: https://www.python.org
✔️ Make sure to enable Add to PATH


---

2. Install VS Code

Install VS Code + Python extension


---

3. Run the Project

Open terminal in VS Code and run:

python resume_app.py


---

🖥️ How It Works

1. Enter personal details 👤


2. Add education, skills, experience 🎓💼


3. Click Generate Resume


4. Resume appears in preview box 📄


5. Click Save Resume to download as .txt file 💾
